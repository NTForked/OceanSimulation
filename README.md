# OceanSimulation
A self-study of simulating ocean waves. Featuring Gerstner waves and Statistic FFT approach. Using OpenGL C++

## Project branch
- master
- DFT_Waves
    
## Installation
Currently support only Windows.

Simply clone the project and open solution file in Visual Studio
